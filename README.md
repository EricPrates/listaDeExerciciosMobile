# Criação de Forms em React Native

Projeto mobile desenvolvido para introduzir conceitos fundamentais de desenvolvimento de aplicações mobile utilizando formulários.

## Tecnologias

- React Native
- Expo Go
- JavaScript

## Objetivo

Introduzir conceitos e tecnologias da disciplina de desenvolvimento de software mobile através da implementação de formulários.

## Conceitos aplicados

- Utilização de **React Hooks**
- Criação de **interfaces de formulário**
- **Validação básica de dados**
- Estruturação de componentes em React Native

## Funcionalidades

- Criação de formulários interativos
- Captura de dados do usuário
- Validação de campos
- Manipulação de estado com Hooks

```bash
npm install
npx expo start --tunnel
